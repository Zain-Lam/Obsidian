# example 1 for limit of a function

TD: YEAR1::TERM2::MA139 Analysis 2::02 Limits and the derivative
Q: Let $f: \mathbb{R} \to \mathbb{R}$ be defined by
$$
f(x) = 
\begin{cases} 
1 & \text{if } x = 0, \\
0 & \text{if } x \neq 0.
\end{cases}
$$
Then what is $\lim_{x \to 0}f(x)$
A: $0$.
