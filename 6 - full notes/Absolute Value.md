# Absolute Value

TD: YEAR1::TERM1::MA141 Analysis 1::Chapter 1
Q: Define the absolute value function
A: $$
|x| =
\begin{cases}
x,  & \text{if $0\leq x$} \\[1ex]
-x, & \text{if $x<0$}
\end{cases}
$$
