# Logically entail

TD: YEAR1::TERM2::PH136 Logic 1
Q: When is a sentence $S$ logically entailed by sentences $P_{1},\dots P_{n}$
A: _iff_ there is no possible situation in which  $P_{1},\dots P_{n}$ are all true and $S$ is false.
